Auto OC 
===================================
Описание
Проект создан для упрощения cозданий ОС связи в Онлайн Школе (Нельзя проверить без аккаунта куратора в Онлайн Школе NeoFamily) 

Стек
openpyxl
textwrap
PyPDF2
os
reportlab
io
requests
datetime


Для начала нужно создать файл datas
Где нужно написать Beers Token. Ссылку с id-аккаунта на NeoFamily, а также данные для входа в аккаунт.
После нужно запустить req.py и ждать. Итогом программы будет excel-таблица с названием: "Data.xlsx" в папке Data. Если вы желаете для ученика добавить уникальную фразу поддержки, то вы можете написать в excel-таблице.
Также вам нужно добавить шаблоны OC связи по каждому предмету.
После всех манипуляцией, вы можете запустить файл shablon.py и снова ждать. Итогом работы будет pdf файлы для всех учеников, файлы находятся в папке Result.
